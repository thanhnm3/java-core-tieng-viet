Generic Class và Method

Generic Class

Khai báo type parameter trong class: class Box<T> { private T value; }. T là type variable — thay thế bằng kiểu cụ thể khi dùng: Box<String>, Box<Integer>. Có thể có nhiều parameter: class Pair<K, V>.

Generic Method

Type parameter khai báo trước return type: <T> T getFirst(List<T> list). Compiler infer từ argument: getFirst(stringList) → T = String. Có thể dùng với class không generic.

// Generic class
class Box<T> {
  private T value;
  public void set(T v) { value = v; }
  public T get() { return value; }
}
Box<Integer> box = new Box<>();
box.set(42);
int n = box.get();  // Không cast
// Generic method
public static <T> T getFirst(List<T> list) {
  return list.isEmpty() ? null : list.get(0);
}
String s = getFirst(List.of("a", "b"));  // T inferred = String
