Modern Java Concurrency

CompletableFuture (Java 8)

Lập trình bất đồng bộ theo phong cách Functional. Kết chuỗi: "Làm A xong → lấy kết quả làm B → lỗi thì C". thenApply, thenCompose, thenCombine, exceptionally.

CompletableFuture<String> future = CompletableFuture
    .supplyAsync(() -> fetchData())
    .thenApply(s -> process(s))
    .exceptionally(ex -> "fallback");
String result = future.get();

Fork/Join Framework

Chia nhỏ tác vụ lớn thành tác vụ con, chạy song song. Thuật toán work-stealing: thread rỗi "đánh cắp" task từ thread bận. ForkJoinPool, RecursiveTask, RecursiveAction. Dùng trong parallelStream().

Virtual Threads (Java 21)

Luồng siêu nhẹ do JVM quản lý, không phải OS. Có thể chạy hàng triệu thread trên laptop. Blocking I/O không tốn OS thread. Thread.ofVirtual().start(task) hoặc Executors.newVirtualThreadPerTaskExecutor().

Platform Thread vs Virtual Thread
Tiêu chí Platform Thread Virtual Thread
Quản lý OS JVM
Stack ~1MB KB, mở rộng khi cần
Số lượng Hàng nghìn Hàng triệu

Khi nào dùng Virtual Threads?

Nên dùng: Ứng dụng nhiều I/O blocking; server HTTP; thay thế thread pool cho task I/O-bound.

Tránh khi: CPU-bound thuần — platform thread vẫn tốt; cần pin (synchronized với native).
