File và Buffered Stream

File (java.io) — đại diện path, không mở file. FileInputStream / FileOutputStream — stream byte. FileReader / FileWriter — stream char (mặc định encoding hệ thống).

BufferedInputStream / BufferedOutputStream

Lớp wrapper: giữ buffer nội bộ (mặc định 8KB). read() lấy từ buffer; khi buffer rỗng mới gọi in.read() xuống đĩa. Giảm số lần system call — đọc 1KB từng byte = 1024 lần; với buffer = 1 lần.

Độ phức tạp — Unbuffered vs Buffered
Thao tác FileInputStream BufferedInputStream
read() 1 byte 1 system call Đọc block, lấy từ buffer
read(byte[]) 8KB 1 system call 1 system call (buffer size)
Đọc 1MB Nhiều call ~128 call (8KB buffer)

GZIPInputStream / GZIPOutputStream

Nén/giải nén GZIP. Wrap stream gốc. Dùng khi cần giảm dung lượng file (log, backup).

// Ghi file nén
try (GZIPOutputStream gz = new GZIPOutputStream(
    new FileOutputStream("data.gz"))) {
  gz.write("Hello".getBytes(StandardCharsets.UTF_8));
}

// Đọc file nén
try (GZIPInputStream gz = new GZIPInputStream(
    new FileInputStream("data.gz"))) {
  byte[] buf = new byte[1024];
  int n;
  while ((n = gz.read(buf)) != -1) {
    System.out.print(new String(buf, 0, n));
  }
}

ByteArrayInputStream / ByteArrayOutputStream

Stream trong memory — dữ liệu từ byte[]. Dùng khi cần đọc/ghi qua API stream nhưng dữ liệu ở memory. ByteArrayOutputStream.toByteArray() lấy byte[] đã ghi.

byte[] data = "Hello".getBytes(StandardCharsets.UTF_8);
try (InputStream in = new ByteArrayInputStream(data)) {
  processStream(in);  // API nhận InputStream
}

ByteArrayOutputStream out = new ByteArrayOutputStream();
out.write("Hello".getBytes());
byte[] result = out.toByteArray();

PushbackInputStream

Cho phép "đẩy lại" byte đã đọc vào buffer. Dùng khi cần peek (xem trước) hoặc parse (đọc thừa cần trả lại).

try (PushbackInputStream in = new PushbackInputStream(
    new FileInputStream("data.txt"), 2)) {
  int b = in.read();
  if (b == 'A') {
    in.unread(b);  // Trả lại để xử lý khác
  }
}

Đọc file text — BufferedReader

// Luôn dùng BufferedReader cho text
try (BufferedReader br = new BufferedReader(
    new InputStreamReader(new FileInputStream("data.txt"), StandardCharsets.UTF_8))) {
  String line;
  while ((line = br.readLine()) != null) {
    System.out.println(line);
  }
}

// Ghi file
try (BufferedWriter bw = new BufferedWriter(
    new OutputStreamWriter(new FileOutputStream("out.txt"), StandardCharsets.UTF_8))) {
  bw.write("Hello");
  bw.newLine();
}

Khi nào dùng Buffered?

Luôn dùng: BufferedInputStream/BufferedOutputStream khi đọc/ghi file; BufferedReader cho text (readLine).

Tránh: FileInputStream/FileReader trực tiếp khi đọc nhiều — chậm do không có buffer.
