Channel — java.nio.channels

Channel là kết nối tới nguồn/đích. Đọc/ghi qua Buffer. FileChannel — file; SocketChannel — TCP; ServerSocketChannel — TCP server; DatagramChannel — UDP.

FileChannel

Lấy từ FileInputStream.getChannel(), FileOutputStream.getChannel(), hoặc RandomAccessFile.getChannel(). read(ByteBuffer) đọc vào buffer; write(ByteBuffer) ghi. transferTo / transferFrom — zero-copy giữa channel (OS hỗ trợ sendfile).

FileChannel vs Stream
Tiêu chí FileInputStream FileChannel
Mô hình Stream byte Buffer + Channel
Zero-copy Không transferTo/transferFrom
Position skip() position(long)
Lock Không lock() / tryLock() 
// Đọc file bằng FileChannel
try (FileChannel channel = FileChannel.open(Path.of("data.txt"), StandardOpenOption.READ)) {
  ByteBuffer buf = ByteBuffer.allocate(8192);
  while (channel.read(buf) != -1) {
    buf.flip();
    process(buf);
    buf.clear();
  }
}

// Copy file — zero-copy
try (FileChannel src = FileChannel.open(Path.of("a.txt"), StandardOpenOption.READ);
     FileChannel dst = FileChannel.open(Path.of("b.txt"),
         StandardOpenOption.WRITE, StandardOpenOption.CREATE)) {
  src.transferTo(0, src.size(), dst);
}

Khi nào dùng FileChannel?

Nên dùng: Copy file lớn (transferTo); cần file lock; random access; tích hợp với Selector (SocketChannel).

Tránh khi: Chỉ đọc file text nhỏ — BufferedReader đơn giản hơn.
