Terminal Operations

Kết thúc pipeline, trigger thực thi và trả về kết quả (hoặc void). Mỗi stream chỉ dùng được một terminal. Short-circuit: findFirst, findAny, anyMatch, allMatch, noneMatch — có thể dừng sớm.

forEach — Duyệt (side-effect)

List.of("a", "b").forEach(System.out::println);

collect — Thu thập vào Collection

// toList (Java 16+)
List<String> list = List.of("a", "b", "c").stream().toList();

// toSet
Set<Integer> set = List.of(1, 2, 2, 3).stream().collect(Collectors.toSet());

// toMap
Map<String, Integer> map = List.of("x", "yy", "zzz").stream()
  .collect(Collectors.toMap(String::toString, String::length));

collectingAndThen — Post-process kết quả

Áp dụng function lên kết quả collector. Ví dụ: collect toList rồi wrap thành unmodifiable.

List<String> unmodifiable = list.stream()
  .collect(Collectors.collectingAndThen(Collectors.toList(), Collections::unmodifiableList));

teeing (Java 12+) — Hai collector cùng lúc

Chạy hai collector trên cùng stream, merge kết quả bằng BiFunction.

// Ví dụ: vừa sum vừa count
var result = List.of(1, 2, 3, 4, 5).stream()
  .collect(Collectors.teeing(
    Collectors.summingInt(Integer::intValue),
    Collectors.counting(),
    (sum, count) -> "sum=" + sum + ", count=" + count
  ));

groupingBy — Nhóm theo key

List<String> words = List.of("apple", "banana", "apricot", "berry");
Map<Character, List<String>> byFirst = words.stream()
  .collect(Collectors.groupingBy(s -> s.charAt(0)));
// {a=[apple, apricot], b=[banana, berry]}

// Với downstream collector
Map<Character, Long> countByFirst = words.stream()
  .collect(Collectors.groupingBy(s -> s.charAt(0), Collectors.counting()));

partitioningBy — Chia 2 nhóm (Predicate)

Map<Boolean, List<Integer>> evensOdds = List.of(1, 2, 3, 4).stream()
  .collect(Collectors.partitioningBy(n -> n % 2 == 0));
// {false=[1, 3], true=[2, 4]}

joining — Nối chuỗi

String joined = List.of("a", "b", "c").stream()
  .collect(Collectors.joining(", "));  // "a, b, c"

String withPrefix = List.of("a", "b", "c").stream()
  .collect(Collectors.joining(", ", "[", "]"));  // "[a, b, c]"

reduce — Gộp thành một giá trị

Hai dạng: reduce(accumulator) trả Optional (stream rỗng → empty); reduce(identity, accumulator) trả T (identity là giá trị mặc định khi rỗng). Identity phải là identity của accumulator: a op identity == a. Với parallel cần combiner: reduce(identity, accumulator, combiner).

Optional<Integer> sum = List.of(1, 2, 3).stream()
  .reduce((a, b) -> a + b);  // Optional[6] — stream rỗng → Optional.empty

Integer sumWithIdentity = List.of(1, 2, 3).stream().reduce(0, Integer::sum);  // 6
// Stream rỗng → 0 (identity)

// SAI — identity không đúng (0 cho multiplication)
// reduce(0, (a,b) -> a*b)  // 0 * x = 0 — sai!
// ĐÚNG: reduce(1, (a,b) -> a*b)

count, min, max

List<Integer> nums = List.of(3, 1, 4, 1, 5);
long count = nums.stream().count();
Optional<Integer> min = nums.stream().min(Integer::compareTo);
Optional<Integer> max = nums.stream().max(Comparator.naturalOrder());

findFirst, findAny, anyMatch, allMatch, noneMatch

Short-circuit — dừng khi có kết quả. findAny hữu ích với parallel (không cần thứ tự). allMatch/noneMatch trên stream rỗng trả true.

Optional<Integer> first = List.of(1, 2, 3, 4).stream().findFirst();
Optional<Integer> any = List.of(1, 2, 3, 4).stream().findAny();  // Hữu ích với parallel

boolean hasEven = List.of(1, 3, 4).stream().anyMatch(n -> n % 2 == 0);
boolean allPositive = List.of(1, 2, 3).stream().allMatch(n -> n > 0);
boolean noNegative = List.of(1, 2, 3).stream().noneMatch(n -> n < 0);

// Stream rỗng
List.<Integer>of().stream().allMatch(x -> false);  // true (vacuous)
List.<Integer>of().stream().anyMatch(x -> true);   // false
