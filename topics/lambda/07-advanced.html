Optional

Container chứa 0 hoặc 1 phần tử — tránh NullPointerException. Không dùng Optional cho parameter, field collection; dùng cho return type khi có thể null.

// Tạo
Optional<String> opt = Optional.of("value");      // Non-null
Optional<String> empty = Optional.empty();
Optional<String> nullable = Optional.ofNullable(getMaybeNull());

// Lấy giá trị
String s = opt.orElse("default");
String s2 = opt.orElseGet(() -> computeExpensive());
opt.ifPresent(System.out::println);

// Chuyển đổi
Optional<Integer> len = opt.map(String::length);
Optional<Integer> flat = opt.flatMap(s -> Optional.of(s.length()));

Optional — Nên và không nên

  • Return type của method có thể trả null
  • Chain: map, flatMap, filter
  • orElse/orElseGet khi cần default
  • Optional làm field, parameter
  • Optional<List> — dùng List rỗng thay vì Optional.empty()
  • opt.get() không kiểm tra — dùng orElse/orElseGet

Parallel Stream

parallelStream() hoặc stream().parallel() — chia công việc cho nhiều thread. Chỉ có lợi khi:

  • Dữ liệu lớn (hàng nghìn phần tử trở lên)
  • Operation tốn CPU, không bị I/O block
  • Không có shared mutable state
// Cẩn thận — reduce cần combiner associative
int sum = list.parallelStream().reduce(0, Integer::sum, Integer::sum);

// Collectors đã thread-safe
Map<String, Long> map = list.parallelStream()
  .collect(Collectors.groupingByConcurrent(Function.identity(), Collectors.counting()));

Parallel Stream — Ưu nhược

  • Dễ dùng, không cần quản lý Executor
  • Overhead với data nhỏ — có thể chậm hơn sequential
  • Shared mutable state → race condition

Một số lỗi thường gặp

Pitfalls
Lỗi Cách tránh
Dùng stream đã consumed Stream chỉ dùng 1 lần — tạo stream mới nếu cần
Side-effect trong filter/map Dùng forEach cho side-effect; filter/map nên pure
Null trong stream filter(Objects::nonNull) hoặc Optional
Boxing overhead (IntStream) Dùng IntStream, LongStream, DoubleStream khi có thể 
// SAI — stream đã closed
Stream<String> s = list.stream();
s.forEach(System.out::println);
s.count();  // IllegalStateException!

// ĐÚNG
list.stream().forEach(System.out::println);
long c = list.stream().count();

Xây dựng bằng Java 17+ · Không dùng framework · Pure NIO
