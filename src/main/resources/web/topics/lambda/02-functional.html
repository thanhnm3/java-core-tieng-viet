Functional Interface

Kỹ thuật

Interface có đúng một abstract method. Dùng @FunctionalInterface để compiler kiểm tra. Java cung cấp sẵn nhiều functional interface trong java.util.function.

Các functional interface cơ bản

java.util.function — các interface chính
Interface Method Mô tả
Predicate<T> boolean test(T t) Kiểm tra điều kiện, trả boolean
Function<T,R> R apply(T t) Chuyển đổi T → R
Consumer<T> void accept(T t) Tiêu thụ T, không trả về
Supplier<T> T get() Cung cấp giá trị, không nhận tham số
UnaryOperator<T> T apply(T t) Function T→T (chuyên biệt)
BiFunction<T,U,R> R apply(T t, U u) Chuyển đổi (T,U) → R
BiPredicate<T,U> boolean test(T t, U u) Điều kiện 2 tham số
BiConsumer<T,U> void accept(T t, U u) Tiêu thụ 2 tham số (Map.forEach)

Primitive variants — Tránh boxing

Khi làm việc với primitive (int, long, double), dùng IntPredicate, LongFunction, DoubleConsumer... để tránh boxing/unboxing overhead.

Primitive functional interfaces
Generic Primitive
Predicate<Integer> IntPredicate, LongPredicate, DoublePredicate
Function<Integer,R> IntFunction<R>, IntToLongFunction, IntToDoubleFunction
Consumer<Integer> IntConsumer, LongConsumer, DoubleConsumer
Supplier<Integer> IntSupplier, LongSupplier, DoubleSupplier

Ví dụ

// Predicate — filter
Predicate<Integer> isEven = n -> n % 2 == 0;
List.of(1, 2, 3, 4).stream().filter(isEven).toList();  // [2, 4]

// IntPredicate — tránh boxing
IntStream.range(0, 10).filter(n -> n % 2 == 0).toArray();

// Function — map
Function<String, Integer> len = s -> s.length();
List.of("a", "ab", "abc").stream().map(len).toList();  // [1, 2, 3]

// Consumer — forEach
Consumer<String> print = s -> System.out.println(s);
List.of("x", "y").forEach(print);

// BiConsumer — Map.forEach
Map.of("a", 1, "b", 2).forEach((k, v) -> System.out.println(k + "=" + v));

// Supplier — lazy init
Supplier<LocalDate> today = LocalDate::now;
LocalDate d = today.get();

Composing — Kết hợp Predicate, Function

// Predicate: and, or, negate
Predicate<Integer> isEven = n -> n % 2 == 0;
Predicate<Integer> gt5 = n -> n > 5;
Predicate<Integer> evenAndGt5 = isEven.and(gt5);

// Function: andThen, compose
Function<Integer, Integer> add1 = x -> x + 1;
Function<Integer, Integer> mul2 = x -> x * 2;
add1.andThen(mul2).apply(3);   // 8: (3+1)*2
add1.compose(mul2).apply(3);    // 7: (3*2)+1

Tự định nghĩa functional interface

@FunctionalInterface
public interface TriPredicate<A, B, C> {
  boolean test(A a, B b, C c);
}

TriPredicate<Integer, Integer, Integer> isTriangle =
  (a, b, c) -> a + b > c && b + c > a && a + c > b;

Khi nào dùng interface nào?

  • Predicate: filter, removeIf, anyMatch, allMatch
  • Function: map, flatMap, computeIfAbsent
  • Consumer: forEach, peek
  • Supplier: Optional.orElseGet, lazy evaluation
  • BiConsumer: Map.forEach, replaceAll
  • IntPredicate, IntFunction: IntStream, tránh boxing
