Intermediate Operations

Các operation trả về Stream, có thể chain. Thứ tự gọi ảnh hưởng hiệu năng — filter trước map để giảm số phần tử cần map. Một số operation stateful (sorted, distinct) cần buffer — ảnh hưởng memory.

filter — Lọc theo điều kiện

List<Integer> evens = List.of(1, 2, 3, 4, 5)
  .stream()
  .filter(n -> n % 2 == 0)
  .toList();  // [2, 4]

map — Chuyển đổi 1-1

List<Integer> lengths = List.of("a", "ab", "abc")
  .stream()
  .map(String::length)
  .toList();  // [1, 2, 3]

mapToInt, mapToLong, mapToDouble — Tránh boxing

Khi kết quả là primitive, dùng mapToInt/mapToLong/mapToDouble để trả về IntStream/LongStream/DoubleStream — tránh boxing overhead, có thêm sum(), average(), summaryStatistics().

// map → Stream<Integer> (boxing)
List.of("a", "ab", "abc").stream().map(String::length).toList();

// mapToInt → IntStream (không boxing)
int sum = List.of("a", "ab", "abc").stream()
  .mapToInt(String::length)
  .sum();  // 6

flatMap — Chuyển đổi 1-nhiều, flatten

Mỗi phần tử map thành Stream, rồi flatten thành một Stream duy nhất.

List<String> words = List.of("hello world", "java stream");
List<String> chars = words.stream()
  .flatMap(s -> Arrays.stream(s.split(" ")))
  .toList();  // ["hello", "world", "java", "stream"]

// List of lists → flat
List<List<Integer>> lists = List.of(List.of(1, 2), List.of(3, 4));
List<Integer> flat = lists.stream()
  .flatMap(List::stream)
  .toList();  // [1, 2, 3, 4]

takeWhile, dropWhile (Java 9+)

takeWhile(predicate): lấy phần tử cho đến khi predicate false. dropWhile(predicate): bỏ phần tử cho đến khi predicate false. Khác filter: dừng khi gặp phần tử không thỏa (với ordered stream).

// takeWhile — dừng khi gặp phần tử không thỏa
List.of(2, 4, 6, 7, 8).stream().takeWhile(n -> n % 2 == 0).toList();
// [2, 4, 6] — dừng ở 7

// dropWhile
List.of(2, 4, 6, 7, 8).stream().dropWhile(n -> n % 2 == 0).toList();
// [7, 8]

sorted — Sắp xếp

Stateful — cần buffer toàn bộ. Với null: Comparator.nullsFirst, Comparator.nullsLast.

// Natural order (Comparable)
List.of(3, 1, 2).stream().sorted().toList();  // [1, 2, 3]

// Custom Comparator
List.of("bb", "a", "ccc").stream()
  .sorted(Comparator.comparingInt(String::length))
  .toList();  // ["a", "bb", "ccc"]

// Với null
List.of("b", null, "a").stream()
  .sorted(Comparator.nullsLast(Comparator.naturalOrder()))
  .toList();  // ["a", "b", null]

distinct, limit, skip

distinct() dùng equals() — stateful, cần lưu đã thấy. Với object custom cần override equals/hashCode. limit(n): short-circuit, không cần xử lý hết. skip(n): bỏ n phần tử đầu.

List.of(1, 2, 2, 3, 3, 3).stream().distinct().toList();  // [1, 2, 3]

Stream.iterate(0, n -> n + 1).limit(5).toList();  // [0, 1, 2, 3, 4]

List.of(1, 2, 3, 4, 5).stream().skip(2).toList();  // [3, 4, 5]

peek — Debug (không thay đổi element)

Dùng để log, debug. Không nên dùng để side-effect vì thứ tự không đảm bảo với parallel. Javadoc: "exists mainly to support debugging".

list.stream()
  .peek(System.out::println)
  .map(String::toUpperCase)
  .toList();
