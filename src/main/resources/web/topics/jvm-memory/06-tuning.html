JVM Tuning — Chi tiết kỹ thuật

Các flag quan trọng khi deploy. Luôn test trên workload thực tế.

Flag Heap và Memory
Flag Ý nghĩa Ví dụ
-Xms Initial Heap size -Xms512m
-Xmx Max Heap size -Xmx2g
-Xmn Young Gen size (Serial/Parallel) -Xmn256m
-Xss Stack size/thread -Xss256k
-XX:MaxMetaspaceSize Giới hạn Metaspace -XX:MaxMetaspaceSize=256m
-XX:MaxDirectMemorySize Direct buffer (NIO) -XX:MaxDirectMemorySize=512m

G1-specific

-XX:MaxGCPauseMillis=200 — mục tiêu pause; -XX:G1HeapRegionSize — kích thước region (1–32MB, power of 2); -XX:InitiatingHeapOccupancyPercent=45 — bắt đầu concurrent mark khi Old đạt 45%.

ZGC-specific

-XX:+UseZGC; -XX:ZCollectionInterval — tần suất GC. ZGC không cần -Xmn — tự quản lý.

Lưu ý deploy

-Xms = -Xmx: Tránh resize Heap — giảm overhead. Container: JVM không nhận diện cgroup limit — dùng -XX:+UseContainerSupport (Java 10+) hoặc set -Xmx thủ công theo limit.

Tránh: -Xmx quá lớn so với RAM vật lý; quên -XX:MaxMetaspaceSize (có thể OOM Native).
