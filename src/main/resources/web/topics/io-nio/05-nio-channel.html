Channel — java.nio.channels

Channel là kết nối tới nguồn/đích. Đọc/ghi qua Buffer. FileChannel — file; SocketChannel — TCP; ServerSocketChannel — TCP server; DatagramChannel — UDP.

FileChannel

Lấy từ FileInputStream.getChannel(), FileOutputStream.getChannel(), hoặc FileChannel.open(Path, options). read(ByteBuffer) đọc vào buffer; write(ByteBuffer) ghi. transferTo / transferFrom — zero-copy giữa channel (OS hỗ trợ sendfile).

SocketChannel — TCP client

SocketChannel.open() tạo channel. connect(SocketAddress) kết nối. configureBlocking(false) cho non-blocking. Đọc/ghi qua ByteBuffer.

try (SocketChannel channel = SocketChannel.open()) {
  channel.connect(new InetSocketAddress("localhost", 8080));
  ByteBuffer buf = ByteBuffer.allocate(1024);
  buf.put("Hello".getBytes(StandardCharsets.UTF_8));
  buf.flip();
  channel.write(buf);
}

ServerSocketChannel — TCP server

ServerSocketChannel.open(), bind(port), accept() trả về SocketChannel. Có thể configureBlocking(false) để dùng với Selector.

ServerSocketChannel server = ServerSocketChannel.open();
server.bind(new InetSocketAddress(8080));
while (true) {
  SocketChannel client = server.accept();  // Blocking
  ByteBuffer buf = ByteBuffer.allocate(1024);
  client.read(buf);
  buf.flip();
  client.write(buf);
  client.close();
}

Scattering read / Gathering write

Scattering: Đọc từ 1 channel vào nhiều buffer tuần tự. Gathering: Ghi từ nhiều buffer ra 1 channel. Hữu ích khi có header + body (buffer 1 = header, buffer 2 = body).

ByteBuffer header = ByteBuffer.allocate(128);
ByteBuffer body = ByteBuffer.allocate(1024);
ByteBuffer[] buffers = { header, body };

// Scattering read
channel.read(buffers);  // Đọc lần lượt vào header, body

// Gathering write
header.flip();
body.flip();
channel.write(buffers);
FileChannel vs Stream
Tiêu chí FileInputStream FileChannel
Mô hình Stream byte Buffer + Channel
Zero-copy Không transferTo/transferFrom
Position skip() position(long)
Lock Không lock() / tryLock() 
// Đọc file bằng FileChannel
try (FileChannel channel = FileChannel.open(Path.of("data.txt"), StandardOpenOption.READ)) {
  ByteBuffer buf = ByteBuffer.allocate(8192);
  while (channel.read(buf) != -1) {
    buf.flip();
    process(buf);
    buf.clear();
  }
}

// Copy file — zero-copy
try (FileChannel src = FileChannel.open(Path.of("a.txt"), StandardOpenOption.READ);
     FileChannel dst = FileChannel.open(Path.of("b.txt"),
         StandardOpenOption.WRITE, StandardOpenOption.CREATE)) {
  src.transferTo(0, src.size(), dst);
}

Khi nào dùng FileChannel?

Nên dùng: Copy file lớn (transferTo); cần file lock; random access; tích hợp với Selector (SocketChannel).

Tránh khi: Chỉ đọc file text nhỏ — BufferedReader đơn giản hơn.
