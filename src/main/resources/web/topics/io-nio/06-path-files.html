Path và Files — NIO.2

Java 7+ có java.nio.file. Path thay File — immutable, cross-platform. Paths.get() hoặc Path.of() (Java 11) tạo Path. Files — utility static: read, write, copy, delete, walk...

Path operations

path.resolve(other) — nối path (path + other). path.relativize(other) — path tương đối từ path đến other. path.normalize() — bỏ . và .. path.toAbsolutePath() — path tuyệt đối.

Path base = Path.of("project", "src");
Path file = base.resolve("main/java/App.java");  // project/src/main/java/App.java
Path rel = base.relativize(file);  // ../main/java/App.java
Path norm = Path.of("a/../b/./c").normalize();  // b/c

FileVisitor — Duyệt cây thư mục

Files.walkFileTree(path, visitor) — duyệt đệ quy, gọi visitor tại mỗi file/dir. SimpleFileVisitor — override visitFile, preVisitDirectory, postVisitDirectory.

Files.walkFileTree(Path.of("src"), new SimpleFileVisitor<>() {
  @Override
  public FileVisitResult visitFile(Path file, BasicFileAttributes attrs) {
    System.out.println(file);
    return FileVisitResult.CONTINUE;
  }

  @Override
  public FileVisitResult preVisitDirectory(Path dir, BasicFileAttributes attrs) {
    if (dir.endsWith("target")) return FileVisitResult.SKIP_SUBTREE;
    return FileVisitResult.CONTINUE;
  }
});

WatchService — Theo dõi thay đổi

FileSystem.getDefault().newWatchService(). path.register(watchService, ENTRY_CREATE, ENTRY_MODIFY, ENTRY_DELETE). watchService.take() block cho đến khi có event.

WatchService watch = FileSystems.getDefault().newWatchService();
Path dir = Path.of("data");
dir.register(watch, StandardWatchEventKinds.ENTRY_CREATE, StandardWatchEventKinds.ENTRY_MODIFY);

while (true) {
  WatchKey key = watch.take();
  for (WatchEvent<?> event : key.pollEvents()) {
    Path name = (Path) event.context();
    System.out.println(event.kind() + ": " + name);
  }
  key.reset();
}

Path vs File

File (java.io) — cũ, có nhiều method I/O; Path — chỉ đại diện path, không mở file. Path dùng với Files hoặc FileChannel. File.toPath() chuyển File sang Path.

Files utility methods
Method Mô tả
Files.readAllBytes(Path) Đọc toàn bộ file
Files.readAllLines(Path, Charset) Đọc lines thành List
Files.write(Path, byte[]) Ghi byte
Files.copy(Path, Path) Copy file
Files.walk(Path) Duyệt cây thư mục (Stream)
Files.exists(Path) Kiểm tra tồn tại 
Path path = Path.of("data", "file.txt");

// Đọc
byte[] bytes = Files.readAllBytes(path);
List<String> lines = Files.readAllLines(path, StandardCharsets.UTF_8);

// Ghi
Files.write(path, "content".getBytes(StandardCharsets.UTF_8));
Files.writeString(path, "content");  // Java 11

// Duyệt thư mục
Files.walk(path, 3)
    .filter(Files::isRegularFile)
    .forEach(System.out::println);

Path + Files — Ưu điểm

Ưu điểm: Immutable; cross-platform; Files utility mạnh; walk, copy, delete atomic; symbolic link.
Lưu ý: readAllBytes/readAllLines load toàn bộ vào memory — không dùng cho file rất lớn.
