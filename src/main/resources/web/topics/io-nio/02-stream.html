InputStream và OutputStream

Đọc/ghi byte. read() trả về byte (0–255) hoặc -1 khi hết; read(byte[] b) đọc vào mảng. write(int b) / write(byte[] b) ghi. Luôn dùng try-with-resources.

Cấu trúc

InputStream là abstract: read() abstract. read(byte[] b) mặc định gọi read() từng byte — chậm. Override trong BufferedInputStream để đọc block. available() ước lượng byte có sẵn (không đáng tin với network).

InputStream vs Reader
Class Đơn vị Encoding Use case
InputStream Byte Không xử lý Binary, image, file
Reader Char (16-bit) Có (InputStreamReader) Text, UTF-8

DataInputStream / DataOutputStream

Đọc/ghi primitive types: readInt(), readDouble(), readUTF()... Data được ghi theo format nhị phân (Java Data Format). Dùng khi cần trao đổi dữ liệu có cấu trúc.

// Ghi
try (DataOutputStream out = new DataOutputStream(
    new BufferedOutputStream(new FileOutputStream("data.bin")))) {
  out.writeInt(42);
  out.writeDouble(3.14);
  out.writeUTF("Hello");
}

// Đọc
try (DataInputStream in = new DataInputStream(
    new BufferedInputStream(new FileInputStream("data.bin")))) {
  int i = in.readInt();
  double d = in.readDouble();
  String s = in.readUTF();
}

ObjectInputStream / ObjectOutputStream — Serialization

Đọc/ghi object (serialization). Class phải implement Serializable. transient — field không serialize. serialVersionUID — version compatibility.

public class Person implements Serializable {
  private static final long serialVersionUID = 1L;
  private String name;
  private transient String password;  // Không ghi ra file
}

try (ObjectOutputStream out = new ObjectOutputStream(
    new FileOutputStream("person.ser"))) {
  out.writeObject(new Person("Alice", "secret"));
}

try (ObjectInputStream in = new ObjectInputStream(
    new FileInputStream("person.ser"))) {
  Person p = (Person) in.readObject();
}

PrintStream / PrintWriter

Ghi text với format. print(), println(), printf(). PrintStream cho byte (System.out); PrintWriter cho char (encoding). PrintWriter không throw IOException — check checkError().

try (PrintWriter pw = new PrintWriter(
    new OutputStreamWriter(new FileOutputStream("out.txt"), StandardCharsets.UTF_8))) {
  pw.println("Line 1");
  pw.printf("Value: %d%n", 42);
}

Đọc byte và text với encoding

// Đọc byte
try (InputStream in = new FileInputStream("data.bin")) {
  byte[] buf = new byte[8192];
  int n;
  while ((n = in.read(buf)) != -1) {
    process(buf, 0, n);
  }
}

// Đọc text với encoding
try (Reader r = new InputStreamReader(
    new FileInputStream("data.txt"), StandardCharsets.UTF_8)) {
  char[] buf = new char[1024];
  int n;
  while ((n = r.read(buf)) != -1) {
    System.out.print(new String(buf, 0, n));
  }
}

Stream — Ưu và nhược điểm

Ưu điểm: Đơn giản; Reader/Writer xử lý encoding; dễ wrap (BufferedInputStream, GZIPInputStream).
Nhược điểm: Blocking; mỗi kết nối cần 1 thread; không có multiplexing.
