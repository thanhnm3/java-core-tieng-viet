Trang chủ

Web Engine hiệu năng cao xây dựng từ Java Core thuần. Khám phá Mạng, Đa luồng và Nội bộ JVM thông qua triển khai Socket thô — không dùng Spring Boot hay framework nào. Website này chính là sản phẩm học tập: mỗi trang bạn đang xem được phục vụ bởi máy chủ HTTP tự viết, chạy trên Java Socket và NIO Selector.

Đoạn mã minh họa

Vòng lặp chính của NioServer — một luồng duy nhất xử lý nhiều kết nối đồng thời nhờ Selector multiplexing.

while (running) {
  int ready = selector.select(1000);
  if (ready == 0) continue;

  Set<SelectionKey> keys = selector.selectedKeys();
  Iterator<SelectionKey> it = keys.iterator();

  while (it.hasNext()) {
    SelectionKey key = it.next();
    it.remove();

    if (key.isAcceptable()) handleAccept(key);
    else if (key.isReadable()) handleRead(key);
    else if (key.isWritable()) handleWrite(key);
  }
}

Ba trụ cột kiến trúc

Engine được xây dựng trên ba lớp kỹ thuật cốt lõi của Java, giúp xử lý hàng nghìn kết nối với tài nguyên tối thiểu.

Tầng 1: I/O không chặn (Non-blocking I/O)

Một luồng duy nhất dùng Selector để đa hợp nhiều kết nối. Không tốn bộ nhớ cho mô hình "một luồng mỗi kết nối".

Tầng 2: Thread Pool tùy chỉnh

Tối ưu chu kỳ CPU bằng các pool Executor chuyên biệt: xử lý request, hoàn thành I/O. Giới hạn số luồng để tránh context switch quá nhiều.

Tầng 3: Routing dựa trên Reflection

Không phụ thuộc framework. Pure Java: ánh xạ đường dẫn tới handler, có thể mở rộng bằng cách đăng ký route mới.
